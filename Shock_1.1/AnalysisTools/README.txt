==========================================================================================
README

This directory contains python tools to read the HDF5 outputs of the PDR code
* extractor:         used to extract quantities from *_s*.hdf5 and *_a*.hdf5 files
* chemistry_analyzer:  used to analyze chemical networks from *_c*.hdf5 files
Version of the tools: 14

Documentations for this two tools are:
* Doc_Extractor.pdf
* Doc_ChemistryAnalyzer.pdf
